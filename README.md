# Chat Platform

Starter real-time group chat app built with React, Vite, Express, and Socket.IO.

## What Is Included

- Frontend: React + Vite chat UI
- Backend: Express + Socket.IO server
- Real-time messaging + typing indicator

## Run Locally

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

- Backend URL: http://localhost:4600
- Frontend URL: http://localhost:3000
